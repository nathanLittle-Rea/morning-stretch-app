# Quick Start - Morning Stretch App

## 30-Second Setup

### Mac or Linux

```bash
cd morning-stretch-app
./launch.sh
```

That's it! Your browser opens automatically.

### Windows

1. Find the `morning-stretch-app` folder
2. Double-click `launch.bat`
3. Browser opens automatically

## First Run

The app installs Python dependencies automatically. First launch takes 30-60 seconds. Afterward, it opens instantly.

## You're ready!

- Click **Start** to begin your 10-minute routine
- Follow the on-screen instructions
- Your progress saves automatically
